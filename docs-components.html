Component Library

All SYX atoms and molecules. Each component is a @mixin called per-theme, using semantic tokens.

How to Use Components

Every SYX component is a SCSS @mixin called per-theme. Components use component tokens for all values Â— never hardcoded colors or sizes.

1. Link the compiled CSS

<link rel="stylesheet" href="css/styles-theme-example-01.css">

2. Add the theme class to body

<body class="syx syx--theme-example-01">

3. Use component classes

<!-- Pattern: .syx-{component} .syx-{component}--{modifier} --> <button class="syx-btn syx-btn--primary syx-btn--filled syx-btn--size-md">Click</button> <input class="syx-input" type="text" placeholder="Enter textÂ…"> <nav class="syx-breadcrumb">Â…</nav>

SCSS Mixin Pattern

// scss/atoms/_btn.scss @mixin atom-btn($theme: null) { .syx-btn { @include transition(color 0.2s ease, background-color 0.2s ease); color: var(--component-btn-primary-color); background: var(--component-btn-primary-bg); @include padding(var(--component-btn-padding-y) var(--component-btn-padding-x)); } }
